# **App Name**: JavaDroid IDE

## Core Features:

- Code Editor: High-performance code editor component optimized for Java. Includes efficient syntax highlighting. Supports accessory row for symbols like ; { } ( ).
- Offline Java Compiler: Bundled Java compiler (JDK/ECJ) within the app for 100% offline code compilation and execution.
- Built-in Terminal: Integrated terminal view to display System.out.print results and manage Scanner inputs.
- Smart Indent: Automated Smart Indent feature for structured code formatting and readability.
- Local File Management: Manage local Java workspace and files using Android's Storage Access Framework.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and focused coding environment.
- Background color: Light grey (#F0F0F0) for a clean and distraction-free interface.
- Accent color: Orange (#FF9800) to highlight interactive elements and important features.
- Code Font: 'Source Code Pro' monospace font for code editing.
- UI font: 'Inter' sans-serif for UI text elements to maintain readability.
- Use clear and simple icons for code actions, file management, and compiler operations.
- Efficient split-screen layout for simultaneous code editing and terminal view.
- Subtle animations for code completion suggestions and compile status updates.
